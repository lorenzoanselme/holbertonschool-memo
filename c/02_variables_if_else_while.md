# 🧮 Variables, Conditions et Boucles

## 🔢 Types de base
`int`, `char`, `float`, `double`, `unsigned`

## ⚙️ Conditions
```c
if (x > 0)
    printf("Positif\n");
else if (x < 0)
    printf("Négatif\n");
else
    printf("Zéro\n");
```

## 🔁 Boucles
```c
for (int i = 0; i < 10; i++)
    printf("%d ", i);

while (n > 0)
    n--;
```

## 🧠 Astuce
- Toujours initialiser les variables.
- Éviter les boucles infinies.
